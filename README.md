# 🛠️ AmneziaWG Config Generator

ابزاری ساده و تحت وب برای تولید و شخصی‌سازی کانفیگ‌های AmneziaWG با استفاده از آی‌پی‌های تمیز کلودفلر.

🔗 **[ورود به برنامه (Live Demo)](لینک_سایت_خود_را_اینجا_بگذارید)**

---

## 💻 قابلیت‌های کلیدی

* 🔍 **آی‌پی‌های تمیز:** دریافت خودکار و پویای آی‌پی‌های تمیز کلودفلر (Warp).
* ⚙️ **تنظیمات پکت‌ها (Junk Packets):** قابلیت تنظیم مقادیر `jc` ،`jmin` و `jmax` در سه حالت سبک، سنگین و دلخواه.
* ⚡ **پشتیبانی از Amnezia 1.5:** امکان فعال‌سازی و مقداردهی پارامترهای پیشرفته آمینزیا (I1-I5).

---

## 🚀 نحوه استفاده

1. وارد لینک وب‌سایت شوید.
2. آی‌پی تمیز مورد نظر خود را انتخاب یا وارد کنید.
3. تنظیمات پکت‌ها و **MTU** (مقدار پیش‌فرض: 1280) را به دلخواه تغییر دهید.
4. تیک فعال‌سازی پارامترهای امنزیا ۱.۵ را روشن کنید.
5. روی دکمه **ساخت کانفیگ** کلیک کرده و فایل خروجی را دانلود یا کپی کنید.

### نحوه وارد کردن کانفیگ در اپلیکیشن:

* **روش کپی:** کانفیگ را کپی کرده، در اپلیکیشن روی علامت **+** بزنید و گزینه **Insert** را انتخاب کنید.
* **روش فایل:** در اپلیکیشن روی علامت **+** بزنید، گزینه **File with connection settings** را انتخاب کرده و فایل دانلود شده را باز کنید.
